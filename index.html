Projects

Here are some of my projects:

  • Machine Learning - Medical Data Analysis using Dynamic Decision Trees
    Implemented a model which automatically learns the hidden patterns in the medical data available from different blogs over the web and identifies the diseases and classify accordingly to facilitate the feature of learning the symptoms accurately and map them to the possible diseases.
  • Machine Learning - Chemical Named Entity Recognition
    Obtaining previous knowledge on chemicals, such as biological properties or toxic effects, can help in many aspects of drug development processes. In this project we discussed various algorithms implemented to extract chemical information and chemical-related entities such as drug names and source materials from text in several domains such as bioinformatics and nanoinformatics. We compared the performances of these algorithms with each other.
  • Information Retrieval - Wikipedia Search engine
    Fetching pages from Wikipedia dump by analyzing the multiple query words, the size of index was less than quarter of the size of Wikipedia dump. Implemented using tokenisation, stemming, inverted indexes, and different degrees of relevancy. It can be used as a search framework wherever we have to deal with a huge data dump.
  • Machine Learning - Sentiment Analysis from Tweets
    Implemented Sentiment Specific Word Embedding (SSWE) model in Torch. Our task is to classify tweets into Negative, Positive and Neutral.
